Permet de nettoyer les données collectées pendant le test pour en exclure les valeurs les plus extrêmes.

Le facteur de conservation permet d'indiquer l'amplitude de conservation.
Soit :

  • m la moyenne des données
  • e l'écart type.
  • f le facteur de conservation
Les données conservées seront celles comprises dans l'interval m - f*e et m + f*e.

Le pourcentage de données à garder permet de limiter le nettoyage en forçant la conservation d'un certain pourcentage de données.
