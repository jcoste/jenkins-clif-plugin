Permet de paramétrer la détection des actions en succés.

Par défaut, seul l'indicateur "success" est utilisé pour déterminer si l'action est en succés ou non.
Pour les actions de type appel HTTP par exemple, ce fonctionnement est trop simpliste car une erreur HTTP 404 est traitée en succés par Clif.
Pour faire une détection plus précise, le plugin peut se baser sur l'analyse du champ "comment" d'une action.
Pour chaque "action type" de Clif, il est possible de définir une expression régulière. Si le champ "comment" matche l'expression régulière alors l'action est considérée comme en succés.

Exemple :
  Pour les actions de type "HTTP GET" on peut définir l'expression régulière ".*OK"
