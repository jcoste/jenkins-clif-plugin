Permet de paramétrer des alias pour les actions mesurées dans Clif.

Par défaut, le nom d'une action est la concaténation des champs "action type" et "comment" de Clif.
Pour faciliter la lecture des rapports, il est possible de définir des alias pour des actions. La définition de ces alias se base sur des expressions régulières.
Si le nom par défaut d'une action matche une expression régulière, le nom de l'action devient alors l'alias associé.
